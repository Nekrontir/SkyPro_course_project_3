# Курсовой проект №3

Данном проекте реализован поиск вакансий с подключением базы данных. <br>
Данные о компаниях и их вакансиях берутся с сайта hh.ru через публичный API и сохраняются в базу данных PostgreSQL. <br>
Программа предоставляет удобный интерфейс для анализа.

## 📋 Функциональность

- Получение информации о заданных компаниях (работодателях) и их вакансиях с hh.ru.
- Автоматическое создание базы данных и таблиц (`employers`, `vacancies`).
- Загрузка данных в БД (с обновлением при повторном запуске).
- Консольное меню для выполнения запросов:
  - Список компаний и количество вакансий.
  - Все вакансии с указанием компании, названия, зарплаты и ссылки.
  - Средняя зарплата по вакансиям.
  - Вакансии с зарплатой выше средней.
  - Поиск вакансий по ключевому слову в названии.

## 🛠 Технологии

- Python 3.13
- [Poetry](https://python-poetry.org/) — управление зависимостями
- requests — взаимодействие с API
- psycopg2-binary — работа с PostgreSQL
- python-dotenv — загрузка переменных окружения


## 📌 Требования к окружению:

Python 3.13 или выше <br>
PostgreSQL (любая современная версия) <br>
Доступ к интернету для обращения к API hh.ru <br>

## 🚀 Установка и запуск

### 1. Клонирование репозитория

`git clone https://github.com/yourusername/hh-parser.git`

### 2. Установка зависимостей через Poetry

*Важно*: Убедитесь, что Poetry установлен, либо установите согласно официальной инструкции.

`poetry install`

Активируйте виртуальное окружение:

`poetry env activate`

### 3. Настройка переменных окружения
Скопируйте файл .env.example в .env и отредактируйте его под свои параметры:

Пример содержимого .env:

```
DB_HOST=localhost 
DB_PORT=5432 
DB_NAME=hh_parser 
DB_USER=postgres 
DB_PASSWORD=your_password 

COMPANY_IDS=00000,000,0000,...
```
*Важно*: Внесите ID компаний для сбора вакансий через запятую, без пробелов. 
ID можно узнать на сайте hh.ru

### 4.Запуск PostgreSQL
Убедитесь, что PostgreSQL запущен и доступен по указанным параметрам. 
База данных hh_parser будет создана автоматически при первом запуске загрузки данных.

### 5. Запуск программы

```python main.py```

В консоли появится меню:
```
--- Меню ---
1. Загрузить данные о компаниях и вакансиях в БД
2. Список всех компаний и количество вакансий
3. Список всех вакансий
4. Средняя зарплата по вакансиям
5. Вакансии с зарплатой выше средней
6. Поиск вакансий по ключевому слову
0. Выход
```
**Рекомендуемый порядок действий**: <br>
Сначала выберите пункт 1, чтобы загрузить данные с hh.ru (займёт некоторое время). <br>
Затем используйте остальные пункты для анализа. <br>
